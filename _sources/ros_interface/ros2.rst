===============
ROS 2 Interface
===============

The Interbotix X-Series LoCoBots are compatible with the following ROS 2 distributions and
platforms:

-   ROS 2 Galactic on Ubuntu Linux 20.04

    -   Create® 3 version **only**

.. -   ROS 2 Humble and Rolling on Ubuntu Linux 22.04

..     -   Create® 3 version **only**
..         -   ROS 2 Humble is the default distribution when ordering from Trossen Robotics.

To install the Interbotix X-Series LoCoBot packages for any of the distributions above, go to the
:doc:`ROS 2 Software Setup Guide<ros2/software_setup>`.

Once installed, follow the steps in the :doc:`ROS 2 Quickstart Guide<ros2/quickstart>` to
get familiar with some of the basic features of the X-Series LoCoBot packages.
