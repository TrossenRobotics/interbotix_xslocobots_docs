===============
ROS 1 Interface
===============

The Interbotix X-Series LoCoBots are compatible with the following ROS 1 distributions and
platforms:

-   ROS 1 Melodic on Ubuntu Linux 18.04

    -   Kobuki version **only**

-   ROS 1 Noetic on Ubuntu Linux 20.04

    -   Both Kobuki and CreateÂ® 3 versions
        -   ROS 1 Noetic is the default distribution when ordering from Trossen Robotics.

To install the Interbotix X-Series LoCoBot packages for any of the distributions above, go to the
:doc:`ROS 1 Software Setup Guide<ros1/software_setup>`.

Once installed, follow the steps in the :doc:`ROS 1 Quickstart Guide<ros1/quickstart>` to
get familiar with some of the basic features of the X-Series LoCoBot packages.
